## 布尔类型

<br>

1) 新的数据类型
* 布尔类型: bool表示布尔量的数据类型
* 取值: 由字面值常量 true (非零)真 false ('\0' NULL false 0) 构成
```c++
bool isok = true;
bool notok = false;
```

2) 字面值true false 可以通过提升转换为int类型 true - 1 false - 0
```c++
int a = true; // a = 1;
```

3) 任何基本类型都可以被隐式转换为布尔类型转换后非0为真 0为假

```c++
#include <iostream>
#include <string>
using namespace std;
bool a;

    cout << a << endl;

    return 0;
```
